## Intro
This is a portfolio website that can help you find a job. To install it you must meet all [Requirements] (## Requirments). As soon as you meet all requirements, you can proceed to [Install] (## Install). Follow the instructions and you are good to go.

## Requirments
* Linux based server with Apache2
* Basic knowledge of WebDesign + Linux commands
* 2MB Free storage
* Root access

## Install

First clone the files into your home folder with

```git clone https://github.com/MikeHermsen/mike-hermsen-gitscrum-port```

This will create an folder called mike-hermsen-gitscrum-port.

Navigate in the file with

```cd mike-hermsen-gitscrum-port/```

next do thing to do is copying all files to the /var/www/html, you can do this with

```cp -r * /var/www/html/.```

To check whether your website is up type

```curl ifconfig.me``` 

This will display an the servers ip address. Type the numbers in your browser search bar. If you see the website loading with the following images then you know it is working correctly.

![Example image of website](https://github.com/MikeHermsen/mike-hermsen-gitscrum-port/examples/homepage.png)
